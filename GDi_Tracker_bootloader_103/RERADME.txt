
1. С начала последней страницы памяти МК загрузчик пытается обнаружить запись в виде структуры  sLoaderRec путем проверки crc на поля структуры и пригодности описанных в ней файлов.
2. Если указанный номер файла не пригоден - проверяется соседний файл и при его пригодности прописывается 0 в поле crc_wrong_flag, передается управление.
3. Если на последней странице не будет обнаружен ни один пригодный файл - reset.



	Настройки загрузчика находятся в файле: loader_defs.h

	В функции void WrTestLoaderRec(u32 addr) подробно расписан порядок заполнения структуры для загрузчика.
	Количество копий структуры на последней странице ограничено размером страницы.
